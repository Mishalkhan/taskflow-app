# TaskFlow App

Live Demo: https://taskflow-app-omega-ten.vercel.app/

## Features
- Add, complete, and delete tasks
- Dashboard with task statistics
- Graphical representation of tasks (completed vs total)
- Summary cards showing task data
- Delete all tasks option in settings
- Landing page with "Get Started" navigation to dashboard

## Tech Stack
- React JS
- Tailwind CSS

## Description
A task management application with a dashboard that visually tracks progress using charts and statistics, helping users manage daily tasks efficiently.
